DESIGN NOTES — OPTION D DOSSIER
Icons: Each action has a hand-drawn SVG icon that references the game's visual language — magnifying glass for Investigate, crossed swords for Fight, boot for Move, running figure for Evade, speech bubble for Parley, coin for Resource. These replace the generic emojis entirely.

Authentic notation: The ► ACTION trigger above each name matches the rulebook's action designator format exactly.

Stat strip: Damage · Horror · Resources · Clues visible at all times with +/− adjusters — no need to navigate away.
